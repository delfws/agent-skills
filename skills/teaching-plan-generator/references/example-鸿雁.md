# 教案示例 - 鸿雁（蒙古族民歌）

## 表头
- 课题：鸿雁
- 课时：1
- 课型：新授
- 备课时间：年月日

## 教学目标
1. 感受蒙古族民歌《鸿雁》的抒情意境，体会思乡情感
2. 萌发对民族音乐的喜爱
3. 通过聆听、模唱等活动，培养音乐感知能力，能哼唱歌曲主题旋律，了解蒙古族民歌的特点及马头琴的相关知识

## 教学分析
- 重点：把握歌曲中速稍慢的抒情风格，学唱主题旋律
- 难点：体会旋律与思乡情感的融合，感受蒙古族民歌韵味
- 教法：情境教学法、聆听模唱法
- 教具：多媒体课件

## 教学过程

| 教学步骤 | 教师活动 | 学生活动 | 设计意图 |
|---------|---------|---------|---------|
| 导入 | 播放草原风光视频，配马头琴音乐，提问："听出这是哪里的声音吗？" 出示歌名《鸿雁》 | 安静聆听，跟随教师走入歌曲情境，齐读歌名 | 快速创设草原情境，引出主题 |
| 聆听感受 | 播放歌曲范唱，提问："歌曲的速度和情绪是怎样的？" 引导学生用词语描述感受 | 聆听并分享感受，初步感知歌曲情绪 | 建立歌曲整体印象 |
| 小结 | 总结：中速稍慢，抒情悠远 | 用词语描述感受 | 建立歌曲整体印象 |
| 歌词赏析 | 出示歌词，提问："找一找歌词里的草原景物，说说你感受到的情感" | 圈画景物，朗读歌词，体会思乡之情 | 结合文字，深化意境理解 |
| 模唱旋律 | 出示主题旋律，用"lu"音带练，分句教唱歌词，伴奏下完整哼唱 | 跟琴模唱，分句学唱，完整哼唱主题 | 实践中掌握旋律，突破教学重点 |
| 文化拓展 | 介绍歌曲背景、马头琴知识，播放马头琴版本片段 | 了解背景，聆听对比，认识马头琴 | 拓展民族文化认知，深化情感体验 |

## 板书设计
```
鸿雁（蒙古族民歌）
速度：中速稍慢
情绪：抒情悠远
情感：思乡之情
乐器：马头琴
```

## 课堂小结
引导回顾：今天我们聆听了草原民歌《鸿雁》，感受了思乡之情，也认识了马头琴。希望大家能爱上我们的民族音乐。

## 教学反思
部分学生长音气息不稳，对思乡情感的理解不够深入。
